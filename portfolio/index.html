YouSession

Descrição

YouSession possibilita aos seus usuários assistir a vídeos do YouTube simultâneamente com demais usuários em um sala. Também fornece interação aos usuários por meio de um chat direto que não necessita login, basta inserir um nickname e pronto.

Tecnologias

Node.js (Javascript) + Html + Css + Materialize.css

EgressoOnline

Descrição

EgressoOnline é uma rede social para os egressos do curso de Sistemas de Informação. Essa rede dá suporte a perfis de usuário, divulgação de emprego e oportunidades, timeline inteligente, envios de mensagens e notificações via email. Também dá suporte a mineração de dados e três tipos diferentes de usuários.

Tecnologias

PHP + Html + Css + Foundation + Javascript + Mysql

CutUrl

Descrição

CurtUrl é um software criado para encurtar url grandes para algo que seja menor. Esse software usa uma tecnologia chamada hash (adler32 algoritmo) que garante que todas as urls encurtadas por ele sejam de apenas um tamanho.

Tecnologias

PHP + Mysql + Html + SlimFramework

Delivery (Webapp & AppMobile)

Descrição

Delivery é uma aplicação que gerencia pedidos. Seu funcionamento básico é enviar a localização (gps) de uma solicitação de entrega para um sistema de gerenciamento de pedidos. Os pedidos são enviados para os prestadores de serviços que estão livres no momento. Esse software se integra com o aplicativo de mensagens Telegram, assim o prestador de serviço pode estar em qualquer lugar e ainda receber pedidos.

Tecnologias

Node.js (Javascript) + Html + Css + Foundation + Telegram API bot + MongoDB + Socket.io + Phonegap
